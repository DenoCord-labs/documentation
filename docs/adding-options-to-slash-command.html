Slash Commands support multiple Option Types:

  • Attachment
  • Boolean
  • Channel
  • Integer
  • Mentionable
  • Number
  • Role
  • String
  • User The Slash commands can also have Sub Command and Sub Command Group for nesting of Slash Command related to each other.

# Adding String Option To Slash Command

// snip

const command = new SlashCommand()
  .setName("ping")
  .setDescription("Pong!")
  .addStringOption((option) => {
    return option
      .setName("name")
      .setDescription("What is Your name?")
      .setRequired(true);
  });

You can also add choices to certain Slash Command Options like String etc.

const command = new SlashCommand()
  .setName("ping")
  .setDescription("Pong!")
  .addStringOption((option) => {
    return option
      .setName("name")
      .setDescription("What is Your name?")
      .setRequired(true)
      .addChoice({
        name: "Unknown",
        value: "Unknown",
      });
  });
