Action rows act as a container for components. An Action row can have only 1 SelectMenu and 5 Buttons at the same time.You can send Maximum 5 action rows in a single message.

Creating an ActionRow:

import { ActionRow } from "https://deno.land/x/denocordts/mod.ts" 


const row  = new ActionRow()

# Methods on Action Row Class

  • addComponents → Takes in instances of Button or SelectMenu.
  • removeAllComponents → Removes all components from ActionRow.
