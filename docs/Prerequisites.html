#Prerequisites

Prerequisites To use DenoCord

  1. Deno must be installed on your machine.
  2. Familiar with Typescript (if you want to use DenoCord with Typescript).
  3. Deno extension for Your Code Editor. You can read more about it here
  4. Knowledge of ES6 syntax. Like async/await, callbacks etc.

If you already know Discord.js it will help you as the api of denocord so similar to discord.js.
