<H2>Prerequisites</H2>
<p> Prerequisites To use DenoCord</p>
</center>

1. [Deno](https://deno.land/) must be installed on your machine.
2. Familiar with [Typescript](https://www.typescriptlang.org/) (if you want to use DenoCord with Typescript).
3. Deno extension for Your Code Editor. You can read more about it [here](https://deno.land/manual/vscode_deno)
Knowledge of [Discord.js](https://discord.js.org/) is not required but will help you as the api of denocord so similar to discord.js.
