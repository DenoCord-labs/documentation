Slash Command can be Created using SlashCommand class.

Example:

import { SlashCommand } from "https://deno.land/x/denocordts/mod.ts";

const command = new SlashCommand().setName("ping").setDescription("Pong");

The name of Slash Command and its options must match this Regex /^[-_\p{L}\p {N}\p{(sc = Devanagari)}\p{(sc = Thai)}]{(1, 32)}$/u

#Registering a Command

There are two types of Slash Commands.

  • Guild Slash Commands
  • Global Slash Commands

#Guild Slash Commands

These commands are best for testing, The appear almost instantly in Discord

#Global Slash Commands

These commands are used when you're making your bot Public, These Commands takes upto an Hour to Show

  • Registering A Global SlashCommand
import { SlashCommand, Client } from "https://deno.land/x/denocordts/mod.ts";

const client = new Client({
  // options
});
const command = new SlashCommand().setName("ping").setDescription("Pong");

await client.registerGlobalSlashCommands([command]);
  • Registering a Guild Command
import { SlashCommand, Client } from "https://deno.land/x/denocordts/mod.ts";

const client = new Client({
  // options
});
const command = new SlashCommand().setName("ping").setDescription("Pong");

await client.registerGuildSlashCommands({
  commands: [command],
  guildId: "762220138321674240", // Guild Id
});
