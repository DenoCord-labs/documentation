What are Slash Commands?

![Slash Commands](/docs/slashCommandFromDiscord.png)

Source: Discord

Slash Commands are commands which consists of a name and description, They show up when users type `/` in their message box. The name and description helps the user to identify the command.

Slash commands can contain options to take input from the Users.

The options are :

`Attachment`,`Boolean`,`Channel`,`Integer`,`Mentionable`,`Number`,`Role`,`String` and `User`.

The Slash commands can also have `Sub Command` and `Sub Command Group` for nesting of Slash Command related to each other.
